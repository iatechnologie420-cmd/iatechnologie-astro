<h2 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_%D9%88%DB%8C%D8%A8_%D8%AE%D8%AF%D9%85%D8%A7%D8%AA%D9%88_%D9%BE%DB%8C%DA%98%D9%86%D8%AF%D9%86%D9%87_AWS_%D9%BE%D9%87_%DA%A9%D9%84%D8%A7%D9%88%DA%89_%DA%A9%D9%85%D9%BE%DB%8C%D9%88%D9%BC%D8%B1%D9%8A_%DA%A9%DB%90_%D8%A7%D9%86%D9%82%D9%84%D8%A7%D8%A8"></span>د ایمیزون ویب خدماتو پیژندنه (AWS): په کلاوډ کمپیوټري کې انقلاب<span class="ez-toc-section-end"></span></h2>



<p>په 2006 کې د هغې د جوړولو راهیسې، <strong>د ایمیزون ویب خدمتونه (AWS)</strong> د بادل خدماتو پلیټ فارم وړاندې کولو سره د آی ټي منظره په بنسټیز ډول بدله کړې چې بې ساري انعطاف ، پیمانه او د پیمانه اقتصاد وړاندې کوي. د دې پیژندنې موخه د عملیاتي اصولو روښانه کول دي<strong>AWS</strong> او دا تشریح کول چې ولې دا حل په کلاوډ کمپیوټري کې کلیدي لوبغاړی شو.</p>



<h3 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_%D9%88%DB%8C%D8%A8_%D8%AE%D8%AF%D9%85%D8%A7%D8%AA_AWS_%DA%85%D9%87_%D8%B4%DB%8C_%D8%AF%DB%8C%D8%9F"></span>د ایمیزون ویب خدمات (AWS) څه شی دی؟<span class="ez-toc-section-end"></span></h3>



<p><strong>AWS</strong> د نړۍ ترټولو پراخه او په پراخه کچه منل شوي کلاوډ کمپیوټري خدماتو پلیټ فارم دی. دا د خدماتو پراخه لړۍ وړاندې کوي چې د معلوماتي ټیکنالوژۍ زیربنا اړتیاوې پوښي، لکه د کمپیوټر بریښنا، د معلوماتو ذخیره کول، او شبکه کول. د AWS خدمتونه د ټولو اندازو سوداګرۍ ته وړتیا ورکوي چې کلاوډ ته لاړ شي یا د بادل کارول پراخه کړي ، نوښت ، ځیرکتیا او وده وړوي.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%D8%B3%D8%B1%D9%87_%D8%AF_%DA%A9%D9%84%D8%A7%D9%88%DA%89_%DA%A9%D9%85%D9%BE%DB%8C%D9%88%D9%BC%D8%B1%D9%8A_%DA%AB%D9%BC%DB%90"></span>د AWS سره د کلاوډ کمپیوټري ګټې<span class="ez-toc-section-end"></span></h4>



<p>د خدماتو کارول <strong>AWS</strong> ډیری ګټې وړاندې کوي. لومړی، د تادیې په څیر موډل د پام وړ لګښت کمولو ته اجازه ورکوي، د معلوماتي ټیکنالوژۍ زیربنا کې د درنو پانګوونو اړتیا له منځه یوسي. لچکتیا او توزیع کول بنسټیز اړخونه دي، د اړتیا سره سم د سرچینو تنظیم کولو وړتیا سره، ستاسو د غوښتنلیکونو لپاره مطلوب فعالیت ډاډمن کول. امنیت هم په لومړیتوب کې دی <strong>AWS</strong>، کاروونکو ته د قوي امنیت چوکاټ او تصدیقونو چمتو کولو سره چې خورا سخت نړیوال معیارونه پوره کوي.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_%D9%88%DB%8C%D8%A8_%D8%AE%D8%AF%D9%85%D8%A7%D8%AA%D9%88_%DA%85%D8%AE%D9%87_%D8%AE%D9%88%D8%B1%D8%A7_%D9%85%D8%B4%D9%87%D9%88%D8%B1_%D8%AE%D8%AF%D9%85%D8%AA%D9%88%D9%86%D9%87"></span>د ایمیزون ویب خدماتو څخه خورا مشهور خدمتونه<span class="ez-toc-section-end"></span></h4>



<p><strong>AWS</strong> د خدماتو بډایه کتابتون وړاندې کوي، مګر ځینې یې د دوی د شهرت لپاره ولاړ دي. د دوی په منځ کې موږ پیدا کوو <strong>ایمیزون EC2</strong> د مجازی سرورونو مدیریت لپاره، <strong>ایمیزون S3</strong> د شیانو ذخیره کولو لپاره، <strong>ایمیزون RDS</strong> د اړونده ډیټابیسونو لپاره <strong>AWS Lambda</strong> د سرور پرته کوډ اجرا کولو لپاره، او <strong>ایمیزون VPC</strong> کوم چې تاسو ته اجازه درکوي یو مجازی خصوصي شبکه جوړه کړئ. د دې خدماتو مدغم کارول د مؤثره او توزیع وړ حلونو رامینځته کول ممکن کوي.</p>



<h2 class="wp-block-heading"><span class="ez-toc-section" id="%D8%A7%D8%B5%D9%84%D9%8A_AWS_%D8%AE%D8%AF%D9%85%D8%AA%D9%88%D9%86%D9%87_EC2%D8%8C_S3%D8%8C_RDS_%D8%A7%D9%88_%D9%86%D9%88%D8%B1"></span>اصلي AWS خدمتونه: EC2، S3، RDS او نور<span class="ez-toc-section-end"></span></h2>

<p>د وړاندیز<strong>د ایمیزون ویب خدمتونه (AWS)</strong> پراخه ده او ځینې وختونه نوي کاروونکو ته پیچلي ښکاري. بیا هم، د لومړني خدماتو پوهیدل کولی شي د AWS بادل منل خورا اسانه کړي. دا مقاله تاسو ته د خورا اړونده AWS خدماتو عمومي کتنه درکوي.</p>



<h3 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%D9%84%DA%86%DA%A9_%D9%88%DA%93_%DA%A9%D9%85%D9%BE%DB%8C%D9%88%D9%BC%D8%B1_%DA%A9%D9%84%D8%A7%D9%88%DA%89_EC2"></span>د AWS لچک وړ کمپیوټر کلاوډ (EC2)<span class="ez-toc-section-end"></span></h3>



<p><strong>AWS EC2</strong> د مجازی مثالونو اداره کولو لپاره بنسټیز خدمت دی. دا کاروونکو ته اجازه ورکوي چې د مجازی کمپیوټر ځواک کرایه کړي او د غوښتنلیک اندازه کولو اداره کړي. EC2 د ترتیب کولو ډیری اختیارونه وړاندیز کوي، د بیلګې په توګه د مختلفو اړتیاو سره تطبیق شوي، ستاسو د خپل عملیاتي سیسټم غوره کولو امکان ته.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%D8%B3%D8%A7%D8%AF%D9%87_%D8%B0%D8%AE%DB%8C%D8%B1%D9%87_%DA%A9%D9%88%D9%84%D9%88_%D8%AE%D8%AF%D9%85%D8%AA_S3"></span>د AWS ساده ذخیره کولو خدمت (S3)<span class="ez-toc-section-end"></span></h4>



<p><strong>S3</strong> د AWS ترټولو مشهور ذخیره خدمت دی. دا د خپل پایښت، شتون او اندازه کولو لپاره مشهور دی. S3 د انځورونو، ویډیوګانو، بیک اپ فایلونو او ډیری نورو معلوماتو ذخیره کولو لپاره کارول کیږي. د دې د څیز جوړښت او د دې مختلف ذخیره کولو ټولګیو څخه مننه ، دا دواړه انعطاف منونکي او اقتصادي دي.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_%D8%AF_%D8%A7%DA%93%DB%8C%DA%A9%D9%88_%DA%89%DB%8C%D9%BC%D8%A7%D8%A8%DB%8C%D8%B3_%D8%AE%D8%AF%D9%85%D8%AA_RDS"></span>د ایمیزون د اړیکو ډیټابیس خدمت (RDS)<span class="ez-toc-section-end"></span></h4>



<p>خدمت <strong>RDS</strong> د اړونده ډیټابیسونو مدیریت ساده کوي. دا د مشهور ډیټابیس انجنونو لکه MySQL، PostgreSQL، اوریکل او SQL سرور ملاتړ کوي. د RDS سره، کاروونکي کولی شي په اسانۍ سره په کلاوډ کې د اړونده ډیټابیس لانچ، چلولو او اندازه کړي.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="AWS_Lambda"></span>AWS Lambda<span class="ez-toc-section-end"></span></h4>



<p><strong>AWS Lambda</strong> یو بې سرور کمپیوټري خدمت دی چې ستاسو کوډ د محرکاتو په ځواب کې پرمخ وړي او په اتوماتيک ډول د اصلي محاسبې سرچینې اداره کوي. لیمبډا اکثرا د پیښې پرمخ وړونکي غوښتنلیکونو رامینځته کولو یا د دندو اتومات کولو لپاره کارول کیږي.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="AWS_%D9%84%DA%86%DA%A9%D8%AF%D8%A7%D8%B1_%D9%84%D9%88%D8%A8%DB%8C%D8%A7"></span>AWS لچکدار لوبیا<span class="ez-toc-section-end"></span></h4>



<p><strong>لچک لرونکي لوبیا</strong> د غوښتنلیک پلي کولو او مدیریت پلیټ فارم دی چې د زیربنا پروسې اتومات کوي لکه د سرچینو چمتو کول ، د بار توازن کول ، اتومات اندازه کول ، او د غوښتنلیک روغتیا نظارت.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_%D8%B3%D8%A7%D8%AF%D9%87_%D8%AE%D8%A8%D8%B1%D8%AA%DB%8C%D8%A7_%D8%AE%D8%AF%D9%85%D8%AA_SNS"></span>د ایمیزون ساده خبرتیا خدمت (SNS)<span class="ez-toc-section-end"></span></h4>



<p><strong>SNS</strong> د پیغام رسولو یو بشپړ مدیریت شوی خدمت دی چې د غوښتنلیک دننه خدماتو ترمینځ ارتباط لپاره ډیزاین شوی. دا د خپرولو / ګډون کولو، د ګرځنده فشار خبرتیاو او خدماتو ته د پیغامونو لیږلو ملاتړ کوي لکه AWS Lambda یا Amazon SQS (د ساده قطار خدمت).</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_%D9%85%D8%AC%D8%A7%D8%B2%DB%8C_%D8%AE%D8%B5%D9%88%D8%B5%D9%8A_%DA%A9%D9%84%D8%A7%D9%88%DA%89_VPC"></span>د ایمیزون مجازی خصوصي کلاوډ (VPC)<span class="ez-toc-section-end"></span></h4>



<p>L&#8217;<strong>ایمیزون VPC</strong> تاسو ته اجازه درکوي د AWS کلاوډ یوه جلا برخه چمتو کړئ چیرې چې تاسو کولی شئ د AWS سرچینې په مجازی شبکه کې پیل کړئ چې تاسو یې تعریف کوئ. دا ستاسو د بادل خدماتو امنیت او شبکې مدیریت لپاره خورا مهم دی.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D8%A7%DB%8C%D9%85%DB%8C%D8%B2%D9%88%D9%86_S3_%DA%AB%D9%84%DB%8C%D8%B4%DB%8C%D8%B1"></span>د ایمیزون S3 ګلیشیر<span class="ez-toc-section-end"></span></h4>



<p><strong>د ایمیزون S3 ګلیشیر</strong> د خورا ټیټ لګښت ذخیره کولو حل دی چې د اوږدې مودې ډیټا آرشیف کولو لپاره ډیزاین شوی. که څه هم د معلوماتو بیا رغونه وخت نیسي، ګلیشیر د ډیټا ذخیره کولو لپاره غوره انتخاب دی چې تاسو اړتیا نلرئ په مکرر ډول لاسرسی ومومئ.</p>

<h3 class="wp-block-heading"><span class="ez-toc-section" id="%D9%BE%D9%87_AWS_%DA%A9%DB%90_%D8%AF_%D8%A7%D9%85%D9%86%DB%8C%D8%AA_%D8%A7%D8%B5%D9%88%D9%84"></span>په AWS کې د امنیت اصول<span class="ez-toc-section-end"></span></h3>



<p><strong>AWS</strong> د ګډ امنیت مفهوم تعقیبولو سره د خپلو پیرودونکو لپاره د لوړې کچې امنیت ساتلو ته ژمن دی. دا پدې مانا ده چې AWS د کلاوډ زیربنا امنیت اداره کوي، پداسې حال کې چې پیرودونکي د دوی د معلوماتو او غوښتنلیکونو ساتلو مسولیت لري. د دې لپاره، AWS مختلف وسایل او تمرینونه وړاندې کوي لکه:</p>



<ul class="wp-block-list">
<li><strong>د پیژندنې او لاسرسي مدیریت (IAM)</strong> : د پیژندنې او لاسرسي مدیریت ترڅو کنټرول شي چې څوک ستاسو د AWS چاپیریال کې څه کولی شي.</li>



<li><strong>Amazon Cognito</strong> : د ګرځنده او ویب غوښتنلیکونو لپاره د خوندي تصدیق او کاروونکي مدیریت خدمت وړاندې کوي.</li>



<li><strong>VPC (مجازی شخصي کلاوډ)</strong> : خدمت تاسو ته اجازه درکوي یو جلا مجازی شبکه رامینځته کړئ ترڅو ستاسو د AWS سرچینې په خوندي ډول ځای په ځای کړي.</li>



<li>د کوډ کولو خدمتونه لکه <strong>د AWS کلیدي مدیریت خدمت (KMS)</strong> او <strong>د AWS سند مدیر</strong> د کلیدي او سند مدیریت لپاره.</li>



<li>د پروګرامونو سره د موافقت چوکاټ لکه GDPR، HIPAA، او FedRAMP.</li>
</ul>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_%D9%81%D8%B9%D8%A7%D9%84%DB%8C%D8%AA_%D9%84%D9%BE%D8%A7%D8%B1%D9%87_%D8%AF_AWS_%D8%AC%D9%88%DA%93%DA%9A%D8%AA_%DA%89%DB%8C%D8%B2%D8%A7%DB%8C%D9%86_%DA%A9%D9%88%D9%84"></span>د فعالیت لپاره د AWS جوړښت ډیزاین کول<span class="ez-toc-section-end"></span></h4>



<p>په AWS کې د لوړ فعالیت جوړښت کې نه یوازې د سرچینو غوره کارول شامل دي بلکه د انعطاف وړ او د توزیع وړ ډیزاین هم شامل دي. AWS خپلولو هڅوي<strong>د ښه معماري چوکاټ جوړښت</strong>، کوم چې په پنځو اړینو ستنو ولاړ دی:</p>



<ol class="wp-block-list">
<li>عملیاتي موثریت</li>



<li>امنیت</li>



<li>اعتبار</li>



<li>فعالیت</li>



<li>د لګښت اصلاح کول</li>
</ol>



<p>دا طریقه د کاروونکو سره مرسته کوي هغه سیسټمونه جوړ کړي چې خورا شتون لري، د غلطۍ زغمونکي، او لګښت- او د فعالیت اغیزمن وي.</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%D8%B3%D8%B1%D9%87_%D8%AF_%D8%A7%D8%B9%D8%AA%D8%A8%D8%A7%D8%B1_%D8%AC%D9%88%DA%93%D9%88%D9%84"></span>د AWS سره د اعتبار جوړول<span class="ez-toc-section-end"></span></h4>



<p>په اړه اعتبار <strong>AWS</strong> د مختلفو کړنو او خدماتو لخوا چمتو کیږي، په شمول:</p>



<ul class="wp-block-list">
<li>د غلطۍ زغمونکي سیسټمونو ډیزاین، لکه د توزیع شوي ډیټابیس خدماتو کارول لکه <strong>Amazon DynamoDB</strong> کوم چې لوړ شتون وړاندې کوي.</li>



<li>د ناکامۍ خطر کمولو لپاره د ډیری شتون زونونو کارول.</li>



<li>د AWS اتومات اندازه کول د ریښتیني وخت غوښتنې پراساس د معلوماتي ټیکنالوژۍ سرچینې تطابق کول او حتی د لوړ بارونو پرمهال هم ثابت فعالیت تضمینوي.</li>



<li>د غوښتنلیک نظارت او مدیریت خدمات لکه <strong>Amazon CloudWatch</strong> او <strong>AWS CloudTrail</strong> د فعالیتونو ریښتیني وخت نظارت او تفصيلي پلټنې لپاره.</li>
</ul>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D9%BE%D9%87_AWS_%DA%A9%DB%90_%D8%AF_%D9%81%D8%B9%D8%A7%D9%84%DB%8C%D8%AA_%D8%A7%D8%B5%D9%84%D8%A7%D8%AD_%DA%A9%D9%88%D9%84"></span>په AWS کې د فعالیت اصلاح کول<span class="ez-toc-section-end"></span></h4>



<p>په بادل کې د فعالیت اصلاح کول پدې معنی دي چې په متحرک ډول د اړتیا سره سم سرچینې تطابق کول. AWS د اصلاح کولو په هدف مختلف خدمتونه وړاندې کوي، لکه:</p>



<ul class="wp-block-list">
<li><strong>د ایمیزون EC2 اتومات اندازه کول</strong> : په اتوماتيک ډول د محاسبې وړتیاوې تنظیم کړئ.</li>



<li><strong>AWS لچکدار بار توازن (ELB)</strong> : د ښه ځواب ویلو او د خطا زغم لپاره د ډیری EC2 مثالونو تر مینځ راتلونکی ترافیک توزیع کول.</li>



<li><strong>ایمیزون S3</strong> او <strong>Amazon CloudFront</strong> : په نړیواله کچه د منځپانګې د چټک او اغیزمن ویش لپاره.</li>



<li>د تحلیل وسیلې لکه <strong>د ایمیزون لچک لټون خدمت</strong> د معلوماتو د چټک تحلیل او لیست کولو لپاره.</li>
</ul>



<h2 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%DA%A9%D9%84%D8%A7%D9%88%DA%89_%DA%AB%D9%BC%DB%90_%D8%A7%D8%AE%DB%8C%D8%B3%D8%AA%D9%86%DB%90_%D9%84%D9%BE%D8%A7%D8%B1%D9%87_%D9%82%D8%B6%DB%8C%DB%90_%D8%A7%D9%88_%D8%BA%D9%88%D8%B1%D9%87_%D8%AA%D9%85%D8%B1%DB%8C%D9%86%D9%88%D9%86%D9%87_%D9%88%DA%A9%D8%A7%D8%B1%D9%88%D8%A6"></span>د AWS کلاوډ ګټې اخیستنې لپاره قضیې او غوره تمرینونه وکاروئ<span class="ez-toc-section-end"></span></h2>

<h3 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%DA%A9%D9%84%D8%A7%D9%88%DA%89_%DA%A9%D8%A7%D8%B1%D9%88%D9%84%D9%88_%D9%82%D8%B6%DB%8C%DB%90"></span>د AWS کلاوډ کارولو قضیې<span class="ez-toc-section-end"></span></h3>



<p>د AWS کلاوډ مختلف خدمتونه وړاندې کوي چې د ډیری کارونې سناریوګانو لپاره مناسب دي، په شمول:</p>



<ul class="wp-block-list">
<li><strong>ذخیره او بیک اپ:</strong> د خوندي شیانو ذخیره کولو یا AWS بیک اپ لپاره ایمیزون S3 وکاروئ ترڅو مرکزي او اتوماتیک بیک اپ وکړئ.</li>



<li><strong>محاسبه:</strong> د سرور پرته پروسس کولو لپاره د Amazon EC2 یا AWS Lambda په کارولو سره د اتوماتیک پیمانه کولو سره غوښتنلیکونه چل کړئ.</li>



<li><strong>ډیټابیس:</strong> د توزیع وړ او مدیریت شوي فعالیت لپاره د Amazon RDS یا Amazon DynamoDB سره ډیټابیسونه کوربه کړئ.</li>



<li><strong>د ناورین بیا رغونه:</strong> د AWS سره د ناورین د بیا رغونې ستراتیژۍ پلان او پلي کړئ.</li>



<li><strong>DevOps:</strong> د AWS CodePipeline او AWS CodeBuild سره د دوامداره ادغام او پلي کولو زنځیرونه پلي کړئ.</li>



<li><strong>د ماشین زده کړه:</strong> د ایمیزون سیج میکر سره د ML ماډلونه جوړ او ځای په ځای کړئ.</li>



<li><strong>د شیانو انټرنیټ (IoT):</strong> د AWS IoT کور سره په لویه کچه د IoT وسایل وصل او اداره کړئ.</li>



<li><strong>د ریښتیني وخت ډیټا جریان:</strong> د Amazon Kinesis سره د ژوندیو معلوماتو جریانونه تحلیل کړئ.</li>
</ul>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%D8%AF_AWS_%DA%A9%D9%84%D8%A7%D9%88%DA%89_%DA%85%D8%AE%D9%87_%DA%AB%D9%BC%D9%87_%D9%BE%D9%88%D8%B1%D8%AA%D9%87_%DA%A9%D9%88%D9%84%D9%88_%D9%84%D9%BE%D8%A7%D8%B1%D9%87_%D8%BA%D9%88%D8%B1%D9%87_%D8%AA%D9%85%D8%B1%DB%8C%D9%86%D9%88%D9%86%D9%87"></span>د AWS کلاوډ څخه ګټه پورته کولو لپاره غوره تمرینونه<span class="ez-toc-section-end"></span></h4>



<p>د AWS بادل څخه د بشپړې ګټې اخیستنې لپاره، دا خورا مهم دي چې غوره عملونه غوره کړئ:</p>



<ul class="wp-block-list">
<li><strong>د معمارۍ پلان جوړونه:</strong> د قوي او موثر سیسټمونو ډیزاین کولو لپاره د AWS ښه معمار شوي چوکاټ وکاروئ.</li>



<li><strong>د لګښتونو مدیریت:</strong> د AWS لګښت اکسپلورر سره لګښتونه وڅارئ او اصلاح کړئ او د لګښتونو خوندي کولو لپاره خوندي یا ځای مثالونه وکاروئ.</li>



<li><strong>امنیت او موافقت:</strong> د امنیت پیاوړي کولو لپاره د AWS وسیلې لکه AWS شناخت او لاسرسي مدیریت (IAM) او Amazon GuardDuty څخه ګټه پورته کړئ.</li>



<li><strong>فعالیت:</strong> د حقیقي اړتیاو سره د منابعو تطابق لپاره د اتوماتیک کولو څخه کار واخلئ او د ټول فعالیت ښه کولو لپاره د Amazon CloudFront مینځپانګې تحویلي شبکه ګټه پورته کړئ.</li>



<li><strong>اتومات کول:</strong> د AWS DevOps وسیلو سره د ادغام او پلي کولو پروسې اتومات کړئ.</li>



<li><strong>اعتبار:</strong> د ډیری شتون زونونو سره د اتوماتیک ناکامۍ میکانیزمونه او د بې ځایه کیدو ستراتیژی پلي کړئ.</li>



<li><strong>نوښت:</strong> د بازار بدلونونو ته په چټکۍ سره د نوښت او ځواب ورکولو لپاره د AWS خدماتو سره په چټکۍ سره تجربه وکړئ.</li>



<li><strong>روزنه او سرچینې:</strong> په پلیټ فارم کې د خپلو مهارتونو د ښه کولو لپاره د AWS اسنادو، روزنې او تصدیقونو څخه ګټه واخلئ.</li>
</ul>



<p>په لنډیز کې، د کارولو قضیو په پوهیدو او د غوره عملونو په غوره کولو سره، سوداګرۍ کولی شي د AWS کلاوډ لخوا وړاندیز شوي د ځواکمن زیربنا او نوښت خدماتو څخه بشپړه ګټه پورته کړي. که د ذخیره کولو، محاسبې، ډیټابیس یا نوښت اړتیاوو لپاره، AWS د سازمانونو د ودې او ډیجیټل بدلون مالتړ لپاره یو تطابق شوی او د اندازې وړ ځواب وړاندې کوي.</p>
