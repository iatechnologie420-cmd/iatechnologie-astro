<h3 class="wp-block-heading"><span class="ez-toc-section" id="1_Creer_un_compte_Google_Cloud"></span>1. Créer un compte Google Cloud<span class="ez-toc-section-end"></span></h3>



<ul class="wp-block-list">
<li>Rendez-vous sur <a href="https://console.cloud.google.com/" rel="nofollow noopener" target="_blank">console.cloud.google.com</a></li>



<li>Créez un nouveau projet (nom libre)</li>



<li>Entrez vos informations de facturation + carte bancaire (<strong>obligatoire mais 100% sécurisée</strong>)</li>
</ul>



<blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow">
<p>ℹ️ Aucune somme ne sera prélevée tant que vous n&#8217;avez pas dépassé vos crédits</p>
</blockquote>



<h3 class="wp-block-heading"><span class="ez-toc-section" id="2_Activer_les_services_requis"></span>2. Activer les services requis<span class="ez-toc-section-end"></span></h3>



<p>Activez dans votre projet :</p>



<ul class="wp-block-list">
<li>Vertex AI</li>



<li>Cloud Storage</li>
</ul>



<p>Vous pouvez le faire ici : <a href="https://console.cloud.google.com/vertex-ai" rel="nofollow noopener" target="_blank">Activer Vertex AI</a> ou <a href="https://console.cloud.google.com/flows/enableapi?apiid=aiplatform.googleapis.com&amp;hl=fr" rel="nofollow noopener" target="_blank">activer l’API Vertex AI</a></p>



<h3 class="wp-block-heading"><span class="ez-toc-section" id="3_Creer_un_bucket_de_stockage"></span>3. Créer un bucket de stockage<span class="ez-toc-section-end"></span></h3>



<ul class="wp-block-list">
<li>Menu ≡ → &#8220;Stockage&#8221; → &#8220;Buckets&#8221;</li>



<li>Créez un nouveau bucket :</li>



<li>Nom : ce que vous voulez (ex: <code>mon-bucket-veo3</code>)</li>



<li>Emplacement : multi-région (us)</li>



<li>Classe : standard</li>
</ul>



<p>Les vidéos Veo seront automatiquement sauvegardées ici.</p>



<h3 class="wp-block-heading"><span class="ez-toc-section" id="4_Creer_une_cle_JSON_de_compte_de_service"></span>4. Créer une clé JSON de compte de service<span class="ez-toc-section-end"></span></h3>



<ul class="wp-block-list">
<li>Menu ≡ → &#8220;IAM et administration&#8221; → &#8220;Comptes de service&#8221;</li>



<li>Créez un compte de service</li>



<li>Allez dans &#8220;Clés&#8221; → &#8220;Ajouter une clé&#8221; → &#8220;Nouvelle clé&#8221; → <strong>JSON</strong></li>



<li>Un fichier <code>.json</code> sera téléchargé. Conservez-le.</li>
</ul>

<ul class="wp-block-list">
<li>Allez sur : <a href="https://console.cloud.google.com/vertex-ai/studio/media" rel="nofollow noopener" target="_blank">Vertex AI Studio &#8211; Media</a></li>



<li>Ouvrez l&#8217;onglet &#8220;Générer&#8221;</li>



<li>Dans le menu à droite :</li>



<li><strong>Modèle</strong> : choisissez &#8220;Veo 3.0&#8221;</li>



<li><strong>Durée</strong> : 8s</li>



<li><strong>Seed</strong> (facultatif) : permet de générer la même scène avec les mêmes paramètres. Pratique pour créer des séries ou un personnage récurrent.</li>
