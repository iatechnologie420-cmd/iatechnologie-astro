<h2 class="wp-block-heading"><span class="ez-toc-section" id="%E3%82%B9%E3%83%86%E3%83%83%E3%83%97_1_%E6%A7%8B%E6%88%90%E3%82%92%E6%BA%96%E5%82%99%E3%81%99%E3%82%8B"></span>ステップ 1: 構成を準備する<span class="ez-toc-section-end"></span></h2>



<p>PS5 でゲームをプレイするときは、Discord で友達と体験を共有するとよいでしょう。これを行うには、PS5 画面を Discord に直接ストリーミングできます。この記事では、DiscordでPS5の画面を共有する方法を2つの方法で説明します。さあ行こう ！</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="%E3%83%8F%E3%83%BC%E3%83%89%E3%82%A6%E3%82%A7%E3%82%A2%E3%82%92%E7%A2%BA%E8%AA%8D%E3%81%97%E3%81%A6%E3%81%8F%E3%81%A0%E3%81%95%E3%81%84"></span>ハードウェアを確認してください<span class="ez-toc-section-end"></span></h4>



<p>始める前に、必要な材料があることを確認してください。必要なものは次のとおりです。<br>&#8211; 機能する PS5: 本体が正常に動作し、テレビに接続されていることを確認してください。<br>&#8211; Discord アカウント: まだアカウントをお持ちでない場合は、Discord の Web サイトにアクセスして無料で登録してください。</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="PC_%E3%81%AB_Discord_%E3%82%BD%E3%83%95%E3%83%88%E3%82%A6%E3%82%A7%E3%82%A2%E3%82%92%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%AB%E3%81%99%E3%82%8B"></span>PC に Discord ソフトウェアをインストールする<span class="ez-toc-section-end"></span></h4>



<p>PS5 の画面を Discord にストリーミングできるようにするには、PC に Discord アプリをインストールする必要があります。その方法は次のとおりです。<br>1. Web ブラウザを開いて、Discord Web サイトにアクセスします。<br>2.「ダウンロード」ボタンをクリックしてアプリケーションをダウンロードします。<br>3. ダウンロードが完了したら、インストーラー ファイルを開き、画面上の指示に従ってアプリケーションを PC にインストールします。</p>



<h4 class="wp-block-heading"><span class="ez-toc-section" id="PS5_%E3%81%AE%E8%A8%AD%E5%AE%9A%E3%82%92%E6%A7%8B%E6%88%90%E3%81%99%E3%82%8B"></span>PS5 の設定を構成する<span class="ez-toc-section-end"></span></h4>



<p>PC に Discord がインストールされたので、Discord へのストリーミング用に PS5 の設定を構成します。従うべき手順は次のとおりです。</p>



<p>1. PS5 の電源を入れ、アカウントにログインします。<br>2. ホーム画面の右上隅にある歯車アイコンを選択して、コンソール設定に移動します。<br>3. 設定で、「キャプチャとブロードキャスト」を選択し、次に「ブロードキャスト設定」を選択します。<br>4. ブロードキャスト設定で、「ブロードキャストを許可する」オプションを有効にします。<br>5. 好みに応じてストリーミング品質を設定します。より良い品質を実現するには、より高速なインターネット接続が必要であることに注意してください。</p>

<p>PS5 の画面を Discord にストリーミングするには、PS Remote Play アプリを使用する必要があります。この無料アプリケーションを使用すると、PS5 をコンピューターまたはスマートフォンにストリーミングできます。 PlayStation の公式 Web サイトにアクセスしてアプリをダウンロードし、インストール手順に従ってください。また、最適な互換性を得るために、PS5 のシステム ソフトウェアが最新バージョンであることを確認してください。</p>
